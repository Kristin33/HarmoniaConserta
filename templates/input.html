The context

* We can do the writing on any key, so feel free to give us key signatures!
e.g: A# --> sharp A major; fb --> flat f minor; G --> G major.
* Since all the chords are of the same length, we don't need the time signature input and set it to 4/4.

The first Chord

* As in the question, the four voices in the first chord is given, so that we can follow the voice leading procedure to write the following chords.
* Also, you need to input the length of the first chord, like 'half', 'quarter'.
* Note: Even we have key signature, do not forget accidentals!
e.g: F2 --> F on the second octave (the bottom line on bass clef!)
G5 --> G on the fifth octave (the highest space on treble clef! )
Db4 --> flat D on the fourth octave.

Harmony -- Roman Numerals

* Drawing a whole roman numeral for us might be hard -- so you can break it down into parts in the input: scale degree, inversion, suspension, secondary.
* Input either number or roman numerals for the scale degree. For example, "2" or "II" or "ii" for the supertonic chord.
* Inversions: 6, 64, 7, 65, 43, 42, or leave it blank!
* If the chord is a secondary chord, indicate which key that it is functioning in. For example: V/IV is the dominant of the subdominant, indicate the subdominant in the "secondary" blank. Either input IV or 4 is okay.
